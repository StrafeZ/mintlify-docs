The Sample Labels tool allows you to quickly generate professional labels for samples, swatches, and mockups. Labels include customer details, reference numbers, and style information in a clean, standardised format.

## Label Format

Each label (2.3" x 1.5") contains the following information:

| Field | Description |
|-------|-------------|
| Customer Name | Company name displayed at the top |
| ATTN | Contact person's name (centered, in dashed box) |
| Date | Label creation date |
| Brand | Customer's brand or parent company |
| Cust Ref | Customer's reference number |
| Our Ref | Your sample number (e.g., SMP-2025-0027) |
| Style | Style or product name |
| Color | Color specification |
| Leather | Material type (default: 100% REAL LEATHER) |
| Comments | Purpose of the sample |

## Creating a Sample Label

<Steps>
  <Step title="Navigate to Export Tools">
    Go to **Shipping â†’ Export Tools** in the sidebar, then click on **Sample Labels**.
  </Step>
  <Step title="Search for your sample">
    Use the search box to find your sample by number, name, or customer. Click on the sample to load its details.
  </Step>
  <Step title="Review and edit details">
    The label details will auto-populate from the sample data. Edit any fields as needed:
    - Update the **ATTN** field with the recipient's name
    - Adjust the **Comments** field based on the purpose
  </Step>
  <Step title="Print or download">
    Click **Print** to send directly to your label printer, or **Download** to save as a PNG image.
  </Step>
</Steps>

## Label Specifications

| Property | Value |
|----------|-------|
| Size | 2.3" x 1.5" (58mm x 38mm) |
| Border | Green (for visibility) |
| Font | System UI (clean, modern) |
| Format | All text uppercase |

<Note>
The green border helps identify Mac Leather labels at a glance. It prints clearly on standard label stock.
</Note>
